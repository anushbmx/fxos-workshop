UI Performance

  • Keep it lean. Think beyond conventions.
  • Custom functions over monolithic libraries.
  • Specific modules over full framework, if necessary
  • Probably you don't need jQuery.
  • Use any UI framework... with caution
  • Avoid excessive DOM manipulations.
  • But, use data attributes wherever possible.
  • Bootstrap or Foundation or Angular or Ionic or rand("ui-framework")?
  • Heavy UI => More power
  • Do more things with Touch events
  • ...but it all depends on your requirement
