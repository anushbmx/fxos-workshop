Device Configuration

  • Enable connecting to the device:
  • Settings -> Developer ->
  • ... Change "Debugging via USB" to "ADB and DevTools"
  • And to prevent disconnecting, keep the screen on:
  • Settings -> Display
  • ... Change "Screen Timeout" to "never"
  • Settings app -> Screen Lock
  • ... Change the toggle to off
